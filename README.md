个人博客（基于Tornado框架）
==========================

依赖
---------

1. Linux
2. Mysql
3. Python 2.6/2.7
4. Tornado 2.4
5. python-mysqldb
6. python-markdown
7. python-pygments


安装
---------

1. 配置好依赖后，运行根目录下的install.py，就能完成博客的初始化
2. 运行blog.py
3. 浏览器中输入对应的地址便可浏览

使用
---------

1. 博主登录地址: /login
2. 博主发文地址: /createArticle
